# CHART OF ACCOUTNS VARIANCE ANALYSIS AUTOMATION

This repository contains a Python automation engine for Variance and Account Movement Analysis within Treasury Reporting. The solution updates Quick Report master data, processes entity and period-based GL activity, links movement data with Balance Sheet Exposure balances, calculates variance logic, and generates Excel-based reporting outputs together with BI-ready datasets. It is designed to improve control, standardization, 
and speed in month-end reporting by minimizing manual intervention and supporting structured analytical review.
